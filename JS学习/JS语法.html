二、JavaScript 变量

变量，顾名思义，一种量可以更变得存在物。变量在编程语言中，通常作为存储数据用。

在 JavaScript 中，无论是什么储存什么类型的数据的变量，统一用关键字：var来声明，用等号来赋值。变量与字面量的区别在于：自声明之后，其值是否可变更。

例如下面这个案例：

var x;
x = 15; // 输出一次
x = 20; // 再输出一次

初始值：x=

重新赋值后：x=

三、其他相关语法概念

1、JavaScript 操作符

主要是一些运算符，有赋值运算符、算术运算符、位运算符、条件运算符、比较运算符和逻辑运算符

2、JavaScript 语句

语句以行为单位，每行语句中可以由关键字、变量或字面量、操作符构成，并不一定要三者同时出现，但至少有一者。

3、JavaScript 关键字

关键字，是一些 JavaScript 的保留字，是有特殊含义和作用的标识。以下 JavaScript 中比较常用的关键字：

4、JavaScript 注释

JavaScript 中使用 “//" 作为语句的注释符，注释符之后的同一行的全部内容都不会执行。
