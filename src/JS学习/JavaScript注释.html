JavaScript学习——JavaScript注释

1、JavaScript的注释符

JavaScript 的注释符和 C/C++ 一样，以双斜杠“//”为单行注释符，以斜杠星号组合“/**/”作为块注释符。形式如下：
// 输出一
。。。。（语句）
// 输出二
。。。。（语句）
/*
......;
......;
*/

2、JavaScript注释的作用

JavaScript注释符的功能就是告诉浏览器那些地方不用执行。
所以它的作用主要有两个：解释代码和阻止执行。
前者主要是为了增加代码的可读性，让其他人知道作者写这句代码的意图或这句代码的功能。
后者就是在某些时刻，如调试的时候，需要暂时不执行某些代码，又不想直接删除代码，因为这样子又可能忘记删除前是怎么写的，
这个时候就可以在代码的前面加上行注释符，或者将代码块用块注释符括起来。
