Javascript学习——Javascript变量

1、何为Javascript变量

Javascript变量和其他编程语言一样，是一种容器，可用于存放值或表达式。
Javascript变量可以是一个的字母，也可以是具有描述性的英文单词，或中文拼音的简拼、全拼。
Javascript变量有以下几个要求：
1、变量名必须以字母开头。
2、变量名不能与关键字一样。
3、变量名对大小写敏感。

2、如何声明使用变量

2.1、声明变量

Javascript中，变量的声明是使用关键字var来声明;
例如：
var carName = "Volvo";
var data = 25;

2.2、使用变量

像2.1中声明了变量后，要怎么使用呢？
变量的使用方式主要有两种：直接输出和与操作符结合参与表达式运算
直接输出的使用形式如下：
var str="我是字符串输出";
document.getElementById("str").innerHTML = str;

值得注意的是，在其他编程语言中如果变量声明了后没有赋值，那么使用的时候会出现乱码或出现错误。
在 Javascript 中，变量声明后，未经赋值便使用的话，程序会自动赋一个“undefined”做为值。
如下：
var temp;
document.getElementById("temp").innerHTML = temp;
