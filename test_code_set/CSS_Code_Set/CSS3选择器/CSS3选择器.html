CSS3选择器
一、CSS3选择器分类

CSS3 选择器在 CSS2.1 的基础上增加了属性选择器、伪类选择器、过滤选择器。
CSS3 选择器可以分为五类：
1）基本选择器
2）组合选择器
3）伪类选择器
4）伪元素选择器
5）属性选择器
其中，伪类选择器又可以细分为6种： 1）动态伪类选择器
2）目标伪类选择器
3）语言伪类
4）UI 元素状态伪类选择器
5）结构伪类选择器
6）否定伪类选择器
